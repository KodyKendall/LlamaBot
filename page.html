Change Your Page with LlamaBot

Learn how to instruct LlamaBot to modify your existing HTML/CSS—step by step.

Step-by-Step Instructions

  1. Provide Context: Paste or describe your current HTML structure so LlamaBot understands where to apply changes.
  2. Specify the Change: Clearly state what you want—for example, “Update the hero section background to a gradient of #ff7e5f → #feb47b.”
  3. Review & Confirm: LlamaBot will return updated HTML. Review it, then integrate it into your page.
  4. Iterate: If you need tweaks, ask follow-up prompts like “Increase the button border radius to 12px.”

Example Prompt

Hi LlamaBot,
Please modify my existing page so that:
1. The header background becomes a dark gradient (#0f2027 → #203a43).
2. The hero section headline reads “Welcome to My New Site!”.
3. Add a drop-shadow under each card in the features section.
4. Increase the font size of all <h3> headings to 1.6rem.

Here’s my current <head> and hero snippet:
<head>…</head>
<section id="hero">…</section>

Tips for Effective Prompts

  • Be concise but precise: mention exact selectors or element IDs.
  • Use hex codes or color names for clarity.
  • Break complex changes into numbered steps.
  • Ask for annotated code snippets so you understand each change.
