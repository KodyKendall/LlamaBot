🦙 LlamaBot v0.3.5 - Screenshot & Video Capture, Slash Commands, and UX Polish

Release Date: February 21, 2026

🚀 What's New

📸 Screenshot & Video Capture ⭐

Powerful new media capture tools for easier bug reporting and communication with Leonardo.

  • Screenshot capture - Take screenshots directly from the chat interface with annotation support
  • Screenshot cropping - Crop screenshots before sending for focused context
  • Drag & drop support - Drag files directly into chat, or copy/paste images
  • Video recording - Record your screen to show Leonardo complex interactions
  • Video attachments for tickets - Attach videos to tickets for richer bug reports
  • Gemini video analysis - Leonardo can watch and analyze attached videos when using Gemini models
  • File upload progress - Visual progress indicator for large file uploads (up to 100MB)

⚡ Slash Commands for Instance Management ⭐

New slash commands for easier Leonardo instance management directly from chat.

  • Database & backup management - View db & active_storage backups from LlamaBot
  • File permissions - Manage file permissions
  • Cron job installation - Install cron jobs via chat
  • Robust /restart - More reliable restart command using systemd-run to survive container restarts
  • Command history modal - View and re-run previous commands

📚 Prompt Library

Save and reuse prompts for common tasks.

  • Save prompts - Store frequently used prompts for quick access

💬 Feedback System

New feedback tab for streamlined user feedback collection.

  • Feedback form - Submit feedback through a dedicated form
  • Easier tracking - Feedback stored in database for better organization

🤖 AI Agent Builder Mode Fix

Fixed AI Agent Builder Mode for creating user-facing agents within Rails apps.

🔧 Improvements

Enhanced Summarization

  • Custom summarization prompt - Comprehensive prompt preserving technical details, code patterns, user intent, and errors/fixes
  • Increased message retention - From 8 to 20 messages to match Claude Code defaults
  • Applied across all agents: rails_agent, rails_testing_agent, rails_ticket_mode_agent

UI/UX Enhancements

  • Resizable chat panel - Drag handle to resize the chat screen (great for screenshots)
  • Selected element button - Moved out of hidden toolbar to be next to chat input
  • Files in message bubbles - Attached files now display in the chat message
  • Relative timestamps - Thread list shows "Just now", "5 minutes ago", "Yesterday", etc.
  • Human-readable thread times - More readable conversation timestamps
  • Dynamic tab title - Tab header shows window URL instead of static "Leonardo"
  • Version tooltip - Hover over Leonardo header to see version
  • User activity tracking - JavaScript timer tracks if user is active in last 15 minutes

WebSocket Reliability

  • WebSocket security - Added security layer for websocket connections
  • Connection error handling - If websocket breaks, shows error and stops thinking mode
  • Security tests - Added tests to verify websocket security

Recording Improvements

  • Smart recording toggle - If already recording rails logs and pressing bug record, it stops the current recording instead of starting a second one

DOM Structure Debugging

  • Nesting doll bug documentation - Documents the pattern of missing closing tags causing JS library failures
  • Console debug script - For detecting DOM nesting issues
  • HTML balance check - Quick Rails console check for HTML structure
  • Testing patterns - Request spec and feature spec patterns for validating HTML structure

Ticket Mode Enhancements

  • HTML vs database tables - Improved prompt to delineate between HTML tables and database tables

Configuration

  • Gemini 3.0 Flash default - Set as the default model
  • .leonardo/LEONARDO.md - Added project-specific context file support
  • Version detection - docker-compose.yml is now the first option to read LlamaBot version

🐛 Bug Fixes

  • Fixed TAVILY_API_KEY initialization using .get() with default empty string (prevents crash if not set)
  • Fixed RuntimeError handling in signal handler registration for pytest/non-main-thread contexts
  • Fixed restart script dying on llamabot takedown - now properly disowns spawned process using nsenter
  • Fixed tooltip version hover on Leonardo h1
  • Fixed AI Agent Builder Mode
